## Privacy Positioning

- No account required.
- No cloud storage.
- No advertising tracking.
- Transfers happen over the local network when using QR upload.
- Files selected by the user are used only for the requested transfer.

## Review Notes Draft

GMP Airdrop is a companion sender app. The iPhone app scans a GMP Airdrop receiver QR code generated by the Windows or Android app, opens the local upload page, and lets the user select files to send over the same Wi-Fi network. The first version does not run a local receive server on iPhone and does not use cloud services.
