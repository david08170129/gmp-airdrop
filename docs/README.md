# GMP Airdrop GitHub Pages Site

This folder contains the static GMP Airdrop landing page.

## Deploy With GitHub Pages

1. Push this repository to GitHub.
2. Open the repository settings.
3. Go to Pages.
4. Set the source to `Deploy from a branch`.
5. Select the branch and choose `/docs` as the folder.
6. Save.

The site is static HTML and CSS only. It does not add a backend and does not change the Flutter app.

## Download Links

The landing page links to static release artifacts in `docs/downloads/`:

- `downloads/GMP_Airdrop_Setup.exe`
- `downloads/GMP_Airdrop_Android.apk`

For a lighter repository, you can replace those links with GitHub Release asset URLs instead of committing binaries.
