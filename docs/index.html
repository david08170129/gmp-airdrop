How to use

Send from iPhone to Windows in a few steps.

  1. Open GMP AirDrop on Windows PC.
  2. Click Wireless Receive.
  3. Make sure iPhone and PC are on the same Wi-Fi.
  4. Open GMP AirDrop on iPhone.
  5. Select photos, videos, or files.
  6. Send to the detected PC or scan the QR code.

Reviewer note

The iOS app is a companion for the Windows receiver. Automatic PC discovery requires the Windows receiver to be open and running on the same Wi-Fi network as the iPhone.

  1. Install and open the Windows receiver.
  2. Start Wireless Receive on Windows.
  3. Open the iOS app on the same Wi-Fi network.
  4. Choose files manually, then send to the detected PC or scan the QR code.

Privacy / local network note

Transfers stay local.

  • No account required.

Troubleshooting

If the iPhone cannot find the PC.

Make sure Wireless Receive is running

Open GMP AirDrop on Windows and click Wireless Receive before searching from iPhone.

Use the same Wi-Fi network

The iPhone and Windows PC must be connected to the same local Wi-Fi network.

Allow Windows Firewall access

If Windows asks for network permission, allow GMP AirDrop on private networks.

Use QR code fallback

If automatic discovery is blocked by the network, scan the QR code shown by the Windows receiver.
