FAQ

Simple answers before you download.

Does iPhone need an app?

No. iPhone can scan the QR code and use Safari for supported browser-based transfers.

Does it use cloud?

No. GMP Airdrop is designed for local transfers only, without cloud storage or accounts.

Does it require same Wi-Fi?

Yes. QR wireless transfer requires the sender and receiver to be on the same local Wi-Fi network.

Is USB-C supported?

Yes. Android to USB-C to Windows is supported for offline drive-based transfer.
